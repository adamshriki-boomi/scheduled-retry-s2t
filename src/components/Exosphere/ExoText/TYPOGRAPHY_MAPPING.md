## Important Notes

1. **Display and Display Bold** are reserved for web/landing pages primarily
2. **Styles are decoupled from `<H1>...<H6>` tags** - semantic HTML should be used separately from visual styling
3. **Links** should apply underline, with/without color (using `--exo-color-font-link`)
4. **Marketing styles** (Display 2, Body Large, Stat) should NOT be used in Boomi Platform UI
5. **Letter spacing** is typically negative for body text (-1% to -2%)
6. **Paragraph spacing** varies by style (16px-22px)

## Style Name to Figma Variable Mapping

All styles in Figma are named with the pattern: `Desktop/[Category]/[Style Name]`

Examples:

- `Desktop/Heading/Headline 1`
- `Desktop/Body/Body`
- `Desktop/Subhead/Subhead 3`
- `Desktop/Code/Code`
